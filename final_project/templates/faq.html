как правильно задать запрос?

запрос может быть любым. буквально. каким угодно.


как выглядит выдача?

выдача состоит из 10 ответов, отранжированных по убыванию соответственно выбранной метрике.


каков размер корпуса?

50000 вопросов и 50000 ответов.


что интересного есть в имплементации?

страничка "нет результатов" для поиска по tf-idf. а еще поиск проводится по вопросам, а не ответам, для большего качества.


с какой скоростью работает поиск?

по обеим метрикам - не больше секунды. при первичном поиске понадобится больше времени, чтобы подгрузились файлы с индексами.


p.s.: копирайт-информация про картинки, потому что интеллектуальный труд - это важно
